Назначение программы
Приложение «Training Planner» — это графическая программа для учёта тренировок. Пользователь может добавлять тренировки с датой, типом и длительностью, фильтровать их по типу или дате, а также сохранять все данные в файл JSON. При следующем запуске данные автоматически загружаются.

Как это работает
Графический интерфейс построен на библиотеке tkinter (встроена в Python).

Поля ввода: дата (формат ГГГГ-ММ-ДД), тип тренировки (текст), длительность в минутах (число).

Кнопка «Добавить тренировку» — добавляет запись в таблицу и сохраняет в JSON.

Два поля для фильтрации (тип и дата) с кнопками «Применить фильтр» и «Сбросить фильтр».

Таблица (Treeview) отображает все тренировки или отфильтрованные.

Проверка корректности ввода:

Дата проверяется функцией validate_date() с помощью datetime.strptime.

Длительность преобразуется в float, проверяется, что она > 0.

При ошибке выводится сообщение через messagebox.showerror().

Фильтрация:

Если введён тип, ищутся тренировки, содержащие эту подстроку (регистронезависимо).

Если введена дата, ищется точное совпадение.

Фильтры можно комбинировать (выведутся записи, соответствующие обоим условиям).

Кнопка «Сбросить фильтр» очищает поля и показывает все записи.

Сохранение и загрузка:

Данные хранятся в файле trainings.json в той же папке, что и программа.

При запуске программа читает JSON и загружает список тренировок.

При добавлении новой тренировки список сразу сохраняется в JSON.

Управление данными:

Каждая тренировка — это словарь с ключами: id (автоинкремент), date, type, duration.

id нужен для уникальной идентификации (хотя в данной версии удаление не реализовано, его легко добавить).
